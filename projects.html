PharmaBot – Autonomous Medicine Delivery

E-Yantra Competition (IIT Bombay)

The PharmaBot project was developed as part of the E-Yantra Robotics Competition organized by IIT Bombay. The objective was to design a robot capable of autonomously delivering medicines from a pharmacy to designated houses using efficient path planning and intelligent control.

  • Path Planning: Implemented Dijkstra’s algorithm to compute shortest routes.
  • Vision: Used OpenCV for detecting road markers suitable for navigation.
  • Controller: Raspberry Pi managed sensors and motors.

Sony SSUP: COWKIN

Smart Cow Health Monitoring System

As part of the Sony Spresense University Program (SSUP), this project encompassed two innovative developments: a Smart Livestock Monitoring System and an Omni Directional Multi-Morphing Robot. The monitoring system utilized the Sony Spresense board to track vital parameters such as body temperature, activity levels, and geolocation of cows.

The Multi-Morphing Robot showcased advanced robotics with shape-shifting capabilities, omni-directional movement, and animal-like leg joints for enhanced adaptability.

Two-Wheeled Inverted Pendulum

Advanced Control Strategies

This project focused on deriving the dynamics of a two-wheeled inverted pendulum and applying advanced control techniques to stabilize it. Dynamics were modeled in MATLAB and tested in MuJoCo.

  • LQR & MPC: Optimal trade-off between control effort and stability.
  • Sliding Mode Control: Robustness against disturbances.
  • Genetic Algorithms: Used for optimizing controller gains.
